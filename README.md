# 🚀 Working Methods To Report Scammer

## 🔹 Method 1: Direct App Report

1. Open scammer profile  
2. Click ⋮ (3 dots)  
3. Tap "Report"  
4. Select reason:
   - Scam  
   - Spam  
   - Fake Account  
5. Submit  

---

## 🔹 Method 2: Official Email Reporting

📧 abuse@telegram.org  

Include:
- Username / ID  
- Chat screenshots  
- Proof of scam  
- Transaction details (if any)
